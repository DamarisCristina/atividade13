z# README - Meu Portal de Entretenimento Digital

## Descrição da Atividade

Este projeto consiste na criação de uma página web em HTML e CSS chamada **Meu Portal de Entretenimento Digital**.

O objetivo foi desenvolver um portal organizado em categorias de entretenimento, apresentando conteúdos de interesse pessoal divididos em três seções principais:

* **Filmes e Séries**
* **Músicas e Podcasts**
* **Games e Tecnologia**

A atividade também teve como foco a prática de:

* Estruturação em HTML
* Estilização com CSS externo
* Uso de Reset CSS
* Inserção de imagens
* Organização visual de conteúdo
* Aplicação de títulos, subtítulos e resumos

---

## Estrutura do Projeto

### Arquivos utilizados:

* `index.html` → Estrutura principal da página
* `reset.css` → Padronização inicial de estilos
* `ativifafe.css` → Estilização personalizada da atividade
* Pasta de imagens → Contendo todas as imagens usadas no portal

## Objetivo Educacional

Esta atividade permitiu praticar conceitos fundamentais de desenvolvimento web, como:

* Criação de páginas informativas
* Separação entre conteúdo (HTML) e estilo (CSS)
* Uso correto de imagens e atributos
* Melhor organização visual
* Desenvolvimento de criatividade digital

## Aprendizados

* Tags HTML (`h1`, `h2`, `h3`, `img`, `link`)

* Background personalizado
